
# 💰 Personal Finance Tracker



The **Personal Finance Tracker** is a Python-based application designed to help users manage and analyze their daily financial transactions. It allows users to record income and expenses, filter transactions by date, and visualize spending patterns using graphs.

This project demonstrates the use of **Object-Oriented Programming (OOP)**, **file handling (CSV)**, and **data analysis with Pandas and Matplotlib**.

---

## 🚀 Features

* Add income and expense transactions
* Store data in a structured CSV file
* Filter transactions by date range
* Generate financial summaries (income, expenses, savings)
* Visualize data using line graphs
* Input validation for date and amount

---

## 🛠️ Tech Stack

* **Python**
* **Pandas**
* **Matplotlib**
* **CSV (File Handling)**
* **Datetime Module**

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/project-finance-tracker.git
```

2. Navigate to the project folder:

```
cd project-finance-tracker
```

3. Install required libraries:

```
pip install pandas matplotlib
```

4. Run the application:

```
python main.py
```

---

## 🧑‍💻 Usage

1. Choose an option:

   * Add a new transaction
   * View transactions within a date range
   * Exit

2. Enter transaction details:

   * Date (DD-MM-YYYY)
   * Amount
   * Category (income/expense)
   * Description

3. View summary:

   * Total income
   * Total expenses
   * Net savings

4. Optional:

   * View graphical representation of transactions
